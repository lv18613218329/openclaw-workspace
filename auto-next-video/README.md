## 安装教程

### Edge 浏览器安装步骤

#### 第一步：打开扩展管理页面

1. 打开 Edge 浏览器
2. 在地址栏输入：`edge://extensions/` 然后按回车
3. 或者点击右上角 **⋯** → 选择 **扩展** → 点击 **管理扩展**

![打开扩展页面](images/step1-edge.png)

#### 第二步：开启开发者模式

1. 在扩展管理页面左侧，找到 **"开发人员模式"** 开关
2. 点击开启（变成蓝色表示已开启）

> ⚠️ **注意**：这一步是必须的！没有开启开发者模式无法安装本地插件。  
> 详见下方 [为什么要开启开发者模式？](#为什么要开启开发者模式)

#### 第三步：加载插件

1. 点击左侧的 **"加载解压缩扩展"** 按钮
2. 在弹出的文件夹选择窗口中，找到并选择 `auto-next-video` 文件夹
3. 点击 **"选择文件夹"**

![加载插件](images/step3-load.png)

#### 第四步：确认安装成功

1. 安装成功后，扩展列表中会出现 **"视频自动下一集"** 插件
2. 浏览器右上角会出现插件图标 🎬

#### 第五步：开始使用

1. 打开视频课程页面
2. 刷新页面（按 F5）
3. 正常观看视频，播放完成后会自动跳转下一集！

### Chrome 浏览器安装步骤

#### 第一步：打开扩展管理页面

3. 点击 **"选择文件夹"**

打开视频页面，刷新后即可使用！

## 为什么要开启开发者模式？

### 🤔 什么是开发者模式？

开发者模式是浏览器提供的一个功能，允许用户安装未经 Chrome/Edge 官方商店审核的插件。

### 🔐 为什么要开启才能安装？

浏览器默认只允许安装官方商店中的插件，这是为了保护用户安全。

但是：
- 官方商店审核需要时间（几天到几周）
- 官方商店上架需要支付费用（Chrome 开发者账号 $5 美元）
- 个人开发的小工具通常不会上架商店

所以，对于个人使用的小插件，通过开发者模式加载是最简单的方式。

### ⚠️ 开启开发者模式安全吗？

**只要你从可信来源获取插件，就是安全的！**

这个插件：
- ✅ 代码完全开源透明，你可以查看所有代码
- ✅ 只在指定的视频网站运行
- ✅ 不收集、不上传任何个人数据
- ✅ 不访问你的其他网页、不读取你的文件
- ✅ 纯本地运行，不需要网络权限

### 🛡️ 如何确保安全？

如果你担心安全问题，可以：

1. **查看代码** - 打开 `content.js` 文件查看源代码
2. **检查权限** - 在扩展详情页查看插件申请的权限
3. **随时卸载** - 不用了随时可以在扩展管理页面移除

### 📜 这个插件会做什么？

| 行为 | 说明 |
|------|------|
| ✅ 监听视频播放状态 | 检测视频是否播放结束 |

- **零数据收集** - 插件不会收集任何个人信息
- **零网络请求** - 插件不会向任何服务器发送请求
- **零第三方跟踪** - 没有任何广告或跟踪代码

## 使用方法

### 基本使用

1. 安装插件后，打开视频课程页面
2. 刷新页面（按 F5）确保插件生效
3. 正常观看视频即可

### 插件工作流程

```
观看视频 → 视频播放完成 → 等待 2 秒 → 自动跳转下一集 → 自动开始播放
```

### 调试功能（可选）

如果你想查看插件工作状态，可以：

1. 按 `F12` 打开浏览器开发者工具
2. 切换到 **"控制台"（Console）** 标签
3. 查看以 `[自动下一集]` 开头的日志信息

也可以输入以下命令：

```javascript
// 查看当前状态
autoNextVideo.status()

// 列出所有视频
autoNextVideo.listVideos()

// 手动测试跳转
autoNextVideo.test()
```

## 常见问题

### ❓ 安装后不工作怎么办？

**请尝试以下步骤：**

1. **刷新视频页面** - 按 F5 刷新
2. **检查插件是否启用** - 在扩展管理页面确认插件是开启状态
3. **检查网站是否正确** - 插件只在特定的视频网站工作
4. **重新加载插件** - 在扩展管理页面点击刷新按钮

### ❓ 提示"无法加载插件"怎么办？

请确认：
- 选择的是 `auto-next-video` 文件夹本身，不是它的父文件夹
- 文件夹内包含 `manifest.json` 文件
- 开发者模式已开启

### ❓ 为什么要刷新页面？

因为插件是在页面加载时注入的。安装插件后，已经打开的页面需要刷新才能加载插件。

### ❓ 插件会影响其他网站吗？

不会。插件只在配置中指定的视频网站运行，不会影响其他网站。

### ❓ 可以在手机上使用吗？

不可以。手机浏览器不支持安装这种扩展插件。

### ❓ 重装浏览器后需要重新安装插件吗？

是的。因为这是本地安装的插件，不是从商店安装的，重装浏览器后会丢失。建议保留插件文件夹备份。

### ❓ 每次打开浏览器都会提示"禁用开发者模式扩展"？

这是浏览器的安全提醒，点击"保留"即可。如果你想避免这个提示，可以：

- Edge：在 `edge://extensions/` 页面确认插件被允许
- Chrome：暂时没有办法完全关闭这个提醒

### ❓ 我想卸载插件怎么办？

1. 打开扩展管理页面（`edge://extensions/` 或 `chrome://extensions/`）
2. 找到"视频自动下一集"插件
3. 点击 **"移除"** 按钮
