# 📦 快速安装指南

## 安装只需 3 步

### Edge 浏览器

```
第 1 步：打开扩展页面
地址栏输入 → edge://extensions/ → 回车

第 2 步：开启开发者模式
左侧找到 "开发人员模式" → 打开开关

第 3 步：加载插件
点击 "加载解压缩扩展" → 选择 auto-next-video 文件夹
```

### Chrome 浏览器

```
第 1 步：打开扩展页面
地址栏输入 → chrome://extensions/ → 回车

第 2 步：开启开发者模式
右上角找到 "开发者模式" → 打开开关

第 3 步：加载插件
点击 "加载已解压的扩展程序" → 选择 auto-next-video 文件夹
```

---

## 关于开发者模式

### ❓ 为什么需要开启？

浏览器默认只允许安装官方商店的插件。这个插件没有上架商店，所以需要开启开发者模式才能安装。

### ✅ 安全吗？

**安全！** 这个插件：
- 代码完全公开透明
- 不收集任何个人信息
- 只在视频网站运行
- 不访问你的其他数据

### 🔍 如何确认安全？

你可以打开 `content.js` 文件查看全部源代码，没有任何加密或隐藏内容。

---

## 使用方法

1. 打开视频课程页面
2. 按 F5 刷新页面
3. 观看视频，结束后自动跳转下一集！

---

**有问题？打开浏览器控制台（F12）输入 `autoNextVideo.status()` 查看状态**
