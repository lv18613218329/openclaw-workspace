实现方式

1. 创建采集 Skill

我可以帮你创建一个专门的 skill，功能包括：

  • 自动遍历全国31个省市的政府采购信息
  • 提取项目详情并去重
  • 增量更新（只保存新增项目）
  • 支持多种关键词配置

2. 配置 Cron 定时任务

{
  "name": "双高建设数据采集",
  "schedule": { "kind": "cron", "expr": "0 9 * * *" },
  "payload": { 
    "kind": "agentTurn", 
    "message": "运行双高建设采购信息采集任务"
  },
  "sessionTarget": "isolated",
  "delivery": { "mode": "announce" }
}

📊 数据存储方案

方案A：CSV 文件（简单）

  • 每天追加新数据
  • 适合数据量小、快速上手
  • 文件：双高建设_全国_[日期].csv

方案B：SQLite 数据库（推荐）

CREATE TABLE procurement_items (
  id INTEGER PRIMARY KEY,
  title TEXT,
  url TEXT UNIQUE,
  publish_date DATE,
  buyer TEXT,
  agent TEXT,
  region TEXT,
  type TEXT,
  created_at DATETIME
);

方案C：飞书多维表格（协作友好）

  • 你已有飞书集成
  • 可以直接推送到飞书多维表格
  • 方便团队协作和筛选
