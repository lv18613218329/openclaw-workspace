## 一、GeometryUnfolding 项目

### 项目位置
`C:\Users\Administrator\.openclaw\workspace\GeometryUnfolding`

### 项目描述
WPF + HelixToolkit.Wpf 实现几何形状的展开和复原动画效果。

### 功能特性
- 5种几何形状：立方体、正四面体、正八面体、正十二面体、正二十面体
- 展开动画：点击按钮触发平滑展开
- 折叠动画：点击按钮恢复原状
- 动画速度控制：滑块调节
- 交互功能：可直接拖拽面控制展开角度

### 技术栈
- .NET 8.0 Windows
- WPF
- HelixToolkit.Wpf 2.25.0

### 遇到的问题
1. HelixToolkit.Wpf 包与 .NET 8 兼容性警告（不影响运行）
2. 尝试切换到 HelixToolkit.Wpf.SharpDX 但 API 差异太大，回退到原版本
3. 拖拽功能初始不可用，后添加了鼠标事件处理实现拖拽控制角度

### 最终状态

## 二、WpfMongoSync 项目

### 项目位置
`C:\Users\Administrator\.openclaw\workspace\WpfMongoSync`

### 项目描述
MongoDB → 飞书多维表格 同步工具，支持增量同步和定时同步。

### 本次修改内容

#### 字段映射修改
根据用户需求，修改了同步到飞书的字段映射：

| 飞书表格列名 | 数据来源 |
|------------|---------|
| **异常名称** | 界面上配置的事件类型（EventFilter） |
| **时间** | MongoDB 数据中的 `local_time` 字段 |
| **内容** | 完整的原始JSON数据 |

#### 代码变更
1. `SyncService.cs` 新增 `TransformData()` 方法
   - 转换数据格式，只保留三个字段
   - 时间字段自动从毫秒时间戳转换为 `yyyy-MM-dd HH:mm:ss` 格式
   - 内容字段存储完整的格式化JSON

2. `MainWindow.xaml.cs` 更新
   - 传入事件类型参数到同步方法

#### 文档更新
- 更新 README.md，添加字段映射说明
- 添加更新日志

### 技术栈
- .NET 8.0 Windows
- WPF
- MongoDB Driver
- Newtonsoft.Json

### 飞书表格字段要求
用户需要在飞书多维表格中创建以下三个字段：
- `异常名称` - 文本类型
- `时间` - 文本类型
- `内容` - 多行文本类型
