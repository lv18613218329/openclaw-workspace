# 电子白板系统架构设计

## 1. 项目概述

**项目名称**: ElectronicWhiteboard  
**技术栈**: .NET Framework 4.8 + WPF + InkCanvas + Clipper2  
**目标**: 面向课堂教学场景的桌面端白板 + 学科图形编辑器

## 4. 核心类设计

### 4.1 ShapeModel（图元模型）

```csharp
public class ShapeModel
{
    public Guid Id { get; set; }
    public ShapeType ShapeType { get; set; }
    
    // 位置与尺寸
    public double X, Y, Width, Height;
    
    // 变换
    public double RotationAngle { get; set; }
    public double CenterX, CenterY;
    public double ScaleX, ScaleY;
    
    // 样式
    public string FillColor { get; set; }
    public string StrokeColor { get; set; }
    public double StrokeThickness { get; set; }
    
    // 连接点
    public List<ConnectorPoint> Connectors { get; set; }
    
    // 分组
    public Guid? GroupId { get; set; }
    
    // 子图形（分割后）
    public List<ShapeModel> Children { get; set; }
}
```

### 4.2 CanvasCommandService（命令服务）

采用**命令模式**实现撤销/重做：

```csharp
public class CanvasCommandService : ICanvasCommandService
{
    private Stack<ICommand> _undoStack = new Stack<ICommand>();
    private Stack<ICommand> _redoStack = new Stack<ICommand>();
    
    public void MoveShapes(IEnumerable<Guid> shapeIds, double deltaX, double deltaY);
    public void RotateShape(Guid shapeId, double angleDelta, double? centerX, double? centerY);
    public void Undo();
    public void Redo();
}
```

### 4.3 SnapService（吸附服务）

```csharp
public class SnapResult
```

## 5. 迭代计划

| 阶段 | 功能 | 交付物 |
|------|------|--------|
| 1 | 基础图形编辑 | Rectangle/Ellipse/Line + 选中/拖动 |
| 2 | 辅助交互 | 多选 + 吸附 + 对齐辅助线 |
| 3 | 几何分割 | Clipper2 集成 + 图形分割 |
| 4 | 连接系统 | 连接点 + 连接线 |
| 5 | 数学工具 | 坐标系、角度、向量 |
| 6 | 物理工具 | 力、电、光学元件 |
| 7 | 化学工具 | 原子、键、环结构 |

## 6. 技术要点

- **InkCanvas**: 手写层，与图形层互斥（通过 `IsHitTestVisible` 切换）
- **Canvas + ItemsControl**: 图形层渲染，使用 DataBinding
- **Adorner**: 选中框、旋转手柄、辅助线等交互元素
- **Clipper2**: 几何布尔运算（分割、合并）
- **命令模式**: 撤销/重做功能
- **工厂模式**: 创建物理/化学图形
