### 一、文档目标与前提

**目标：**  
在 **.NET Framework 4.8 + WPF + InkCanvas** 技术栈下，设计并实现一套可商用的“板书 + 学科图形编辑器”系统，支持：

- 手写笔记（黑板书写）
- 数学系图形工具
- 物理系常用图形工具
- 化学系常用图形工具

**技术前提：**

- 运行环境：Windows + .NET Framework 4.8
- UI 框架：WPF
- 手写控件：`System.Windows.Controls.InkCanvas`
- 第三方库（均支持 .NET 4.8）：
  - **Clipper2**：几何布尔运算（分割、合并、求交）
  - **WPF-Math**（可选）：数学公式渲染
  - **ScottPlot 4.x**（可选）：函数/实验曲线图
  - **FontAwesome.WPF / Lucide-WPF**（可选）：工具栏图标

---

## 二、总体架构设计

### 2.1 分层架构

**1）UI 层（WPF）**

- **InkCanvas 层：**  
  - 用于板书、手写笔记、批注。
- **图形 Canvas 层：**  
  - 用于显示和操作所有“图形对象”（数学/物理/化学图形）。
- **Adorner 层：**  
  - 用于显示选中框、旋转手柄、辅助对齐线等。

**2）应用层（交互逻辑）**

- 工具系统（当前工具：选择、绘制、旋转、分割、连线等）
- 命令服务（移动、旋转、分割、连接、填充、分组等）
- 状态管理（当前选中图形、当前学科模式等）

**3）核心模型与几何层（Core）**

- 图元模型（`ShapeModel`、`ConnectionModel`）
- 画布状态（`CanvasState`）
- 几何运算（基于 Clipper2）
- 吸附 / 对齐算法

## 三、核心数据模型设计

### 3.1 图元模型（ShapeModel）

```csharp
public enum ShapeType
{
    Rectangle,
    Ellipse,
    Polygon,
    Line,
    Arrow,
    // 数学
    Vector,
    CoordinateSystem,
    // 物理
    ForceArrow,
    Pulley,
    Resistor,
    // 化学
    Atom,
    Bond,
    Ring
}

public class ConnectorPoint
{
    public Guid Id { get; set; } = Guid.NewGuid();
    public double RelativeX { get; set; }  // 相对图形局部坐标
    public double RelativeY { get; set; }
}

public class ShapeModel
{
    public Guid Id { get; set; } = Guid.NewGuid();
    public ShapeType ShapeType { get; set; }

    // 画布坐标系中的位置与尺寸
    public double X { get; set; }
    public double Y { get; set; }
    public double Width { get; set; }
    public double Height { get; set; }

    // 旋转（以 CenterX/CenterY 为中心）
    public double RotationAngle { get; set; }
    public double CenterX { get; set; }
    public double CenterY { get; set; }

    // 填充与描边
    public string FillColor { get; set; } = "#FFFFFFFF";
    public string StrokeColor { get; set; } = "#FF000000";
    public double StrokeThickness { get; set; } = 2.0;

    // 连接点
    public List<ConnectorPoint> Connectors { get; set; } = new List<ConnectorPoint>();

    // 分割后的子图形
    public List<ShapeModel> Children { get; set; } = new List<ShapeModel>();

    // 分组
    public Guid? GroupId { get; set; }
}
```

### 3.2 连接线模型（ConnectionModel）

```csharp
public class ConnectionModel
{
    public Guid Id { get; set; } = Guid.NewGuid();

    public Guid FromShapeId { get; set; }
    public Guid FromConnectorId { get; set; }

    public Guid ToShapeId { get; set; }
    public Guid ToConnectorId { get; set; }

    public bool IsDirected { get; set; }  // 是否箭头（向量、电路等）
}
```

### 3.3 画布状态与命令接口

```csharp
public class CanvasState
{
    public List<ShapeModel> Shapes { get; set; } = new List<ShapeModel>();
    public List<ConnectionModel> Connections { get; set; } = new List<ConnectionModel>();

    public List<Guid> SelectedShapeIds { get; set; } = new List<Guid>();
}

public interface ICanvasCommandService
{
    void AddShape(ShapeModel shape);
    void RemoveShape(Guid shapeId);
    void MoveShapes(IEnumerable<Guid> shapeIds, double deltaX, double deltaY);
    void RotateShape(Guid shapeId, double angleDelta, double? centerX = null, double? centerY = null);
    void SplitShape(Guid shapeId, /* 分割线参数 */);
    void GroupShapes(IEnumerable<Guid> shapeIds);
    void Ungroup(Guid groupId);
    void SetFillColor(Guid shapeId, string color);
}
```

## 五、功能级技术实现指导

下面逐条对应你的业务需求，给出技术实现思路。

---

### 5.1 数学系工具

#### 5.1.1 图形以某一点进行旋转

**需求：**

- 任意图形可以围绕任意一点旋转（默认是图形中心）。

**实现要点：**

- 在 `ShapeModel` 中维护 `RotationAngle`、`CenterX`、`CenterY`。
- WPF 端使用 `RotateTransform` 绑定这三个属性。
- 使用 `Adorner` 实现旋转手柄。

**旋转 Adorner 逻辑：**

- 在选中图形上方绘制一个小圆点（`Thumb`）。
- 鼠标按下时记录：
  - 旋转中心点（`CenterX/CenterY`）
  - 起始向量：`startVector = startPoint - center`
- 鼠标移动时：
  - 当前向量：`currentVector = currentPoint - center`
  - 角度变化：
    ```csharp
    var angleDelta = Vector.AngleBetween(startVector, currentVector);
    _canvasCommandService.RotateShape(shape.Id, angleDelta);
    ```
- 鼠标抬起时结束。

---

#### 5.1.2 图形上有连接点，可与其他图形连接

**需求：**

- 图形上有“连接点”，可以用线连接，适用于向量、几何构造、电路、化学键等。

**实现要点：**

- 在 `ShapeModel.Connectors` 中定义连接点（相对坐标）。
- 渲染时，根据图形位置和旋转计算连接点的屏幕坐标。
- 连接线使用 `ConnectionModel` 表示。

**交互流程：**

1. 用户选择“连线工具”。
2. 在某个图形的连接点附近点击 → 识别最近连接点 → 作为起点。
3. 鼠标移动时绘制临时线。
4. 鼠标释放时：
   - 若命中另一个连接点 → 创建 `ConnectionModel`。
   - 否则取消。

---

#### 5.1.3 图形之间相互吸靠（对齐 / 吸附）

**需求：**

- 图形拖动时，自动对齐到其他图形的中心线、边缘线，显示辅助线。

**实现要点：**

- 在 Core 层实现 `ISnapService`：
  - 输入：当前画布状态、正在移动的图形 Id、目标位置。
  - 输出：吸附后的 X/Y + 辅助线列表。

**算法示意：**

- 对每个其他图形：
  - 计算其中心 X/Y、Left/Right/Top/Bottom。
- 对当前图形：
  - 计算其中心 X/Y、Left/Right/Top/Bottom。
- 若某一对线之间的距离 < 阈值（如 5 像素）：
  - 将当前图形的对应线吸附到目标线。
  - 记录一条辅助线（用于前端绘制）。

**前端：**

- 拖动时调用 `ISnapService.GetSnappedPosition(...)`。
- 用返回的 `SnappedX/Y` 更新图形位置。
- 用返回的辅助线在 `Adorner` 上绘制。

---

#### 5.1.4 图形内部可分割，分割后子图形可单独旋转

**需求：**

- 图形内部可以被“切割”，切割后形成多个子区域，每个子区域可以单独旋转、填色。

**实现要点：**

- 使用 **Clipper2** 做几何布尔运算。

**步骤：**

1. 将原图形转换为多边形点集（`List<IntPoint>`）。
2. 将用户绘制的“分割线”转换为一个细长多边形（或直接用 Clipper 的线段分割逻辑）。
3. 使用 Clipper2 的布尔运算：
   - `Difference` / `Intersection` 等，得到多个子多边形。
4. 为每个子多边形创建一个新的 `ShapeModel`，挂到原图形的 `Children` 中。

**伪代码：**

```csharp
public void SplitShape(Guid shapeId, PolylineModel cuttingLine)
{
    var shape = FindShape(shapeId);
    var polygon = ShapeToPolygon(shape);
    var cutter = PolylineToPolygon(cuttingLine);

    var resultPolygons = Clipper2Wrapper.SplitPolygon(polygon, cutter);

    shape.Children.Clear();
    foreach (var poly in resultPolygons)
    {
        var child = ShapeModelFactory.FromPolygon(poly);
        child.GroupId = shape.GroupId;
        shape.Children.Add(child);
    }
}
```

**前端：**

- 若 `Children.Count == 0` → 渲染原图形。
- 若 `Children.Count > 0` → 渲染子图形。
- 子图形可以：
  - 单独选中
  - 单独旋转（同样使用 `RotateTransform`）
  - 单独填充颜色

---

#### 5.1.5 图形区域填充颜色

**需求：**

- 图形或子图形区域可以填充颜色。

**实现要点：**

- `ShapeModel.FillColor` 绑定到 WPF `Shape.Fill`。
- 工具栏提供颜色选择器：
  - 选中图形 → 点击颜色 → 调用 `SetFillColor(shapeId, color)`。

---

#### 5.1.6 多选拖动 + 等距辅助线 / 水平辅助线

**需求：**

- 多个图形可以一起选中、一起拖动。
- 拖动或布局时有等距辅助线、水平/垂直对齐线。

**实现要点：**

- `CanvasState.SelectedShapeIds` 维护选中集合。
- 框选：
  - 鼠标拖出一个矩形，HitTest 所有图形 Bounds。
- 拖动：
  - 鼠标按下时记录起点。
  - 移动时计算 `deltaX/Y`，调用：
    ```csharp
    MoveShapes(SelectedShapeIds, deltaX, deltaY);
    ```
- 等距辅助线：
  - 提供一个“平均分布”命令：
    - 计算选中图形的最小/最大 X/Y。
    - 按数量等分间距，重新设置每个图形的位置。
  - 或在拖动时显示与其他图形的间距。

### 5.2 物理系工具

**需求：**

- “基本老师常用工具”：力学、电学、光学常见图形。

**建议图元：**

- 力学：
  - 力的箭头（带大小、方向标注）
  - 斜面、小车、滑轮、弹簧
- 电学：
  - 电源、电阻、电容、电感、开关、导线
- 光学：
  - 光线、平面镜、凸透镜、凹透镜、焦点标记

**实现方式：**

- 在 Core 层定义 `PhysicsShapeFactory`：

```csharp
public static class PhysicsShapeFactory
{
    public static ShapeModel CreateForceArrow(Point start, Point end) { ... }
    public static ShapeModel CreatePulley(Point center) { ... }
    public static ShapeModel CreateResistor(Point position) { ... }
}
```

- 每个物理元件：
  - 用 `ShapeModel.ShapeType` 标识类型。
  - 在 WPF 渲染时，根据类型选择不同的 PathGeometry 或自定义控件。
- 电路元件端点：
  - 定义为 `ConnectorPoint`，用连接线系统连接导线。

---

### 5.3 化学系工具

**需求：**

- “基本老师常用工具”：原子、键、环结构等。

**建议图元：**

- 原子：C、H、O、N 等
- 键：单键、双键、三键
- 环：苯环、五元环等

**实现方式：**

- 在 Core 层定义 `ChemistryShapeFactory`：

```csharp
public static class ChemistryShapeFactory
{
    public static ShapeModel CreateAtom(string symbol, Point position) { ... }
    public static ShapeModel CreateBond(Guid atom1Id, Guid atom2Id, int order) { ... }
    public static ShapeModel CreateBenzeneRing(Point center, double radius) { ... }
}
```

- 原子：
  - `ShapeType.Atom`，渲染为一个圆 + 文本（元素符号）。
- 键：
  - 可以用 `ConnectionModel` 表示（From/To 为原子）。
  - 单键：一条线；双键：两条平行线；三键：三条线。
- 环：
  - 预定义正多边形（六边形等），作为一个整体 `ShapeModel`。

## 六、InkCanvas 与图形层的协同

### 6.1 模式切换

- **书写模式：**
  - `InkCanvas.IsHitTestVisible = true`
  - `ShapeLayer.IsHitTestVisible = false`
- **图形模式：**
  - `InkCanvas.IsHitTestVisible = false`
  - `ShapeLayer.IsHitTestVisible = true`

### 6.2 交互体验建议

- 工具栏上有“笔 / 橡皮 / 图形工具 / 选择工具”等按钮。
- 当前模式在 UI 上有明显高亮提示。
- 支持快捷键切换（如：`P`=笔，`S`=选择，`G`=图形）。

## 七、迭代建议（按优先级）

1. **第一阶段：基础图形编辑**
   - 添加矩形/圆/线段

2. **第二阶段：辅助交互**
   - 多选

4. **第四阶段：连接系统**
   - 连接点 + 连接线
   - 支持向量、电路、化学键

5. **第五阶段：学科工具**
   - 数学工具栏
   - 物理工具栏
