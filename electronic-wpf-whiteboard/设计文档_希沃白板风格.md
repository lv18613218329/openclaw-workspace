# 电子白板界面设计文档 (希沃白板风格) - V2

## 1. 设计目标

参考希沃白板5(Seewo)的界面风格和功能布局，重新设计 Work Agent 的电子白板界面。**操作按钮集中在底部**，支持二级/三级菜单展开。

## 3. 底部工具栏设计 (高度: 180px)

### 3.1 工具栏分层结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│  一级菜单 tabs                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│  二级菜单 + 三级菜单内容区                                              │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 一级菜单 (主分类)

| 按钮 | 功能说明 |
|------|----------|
| 画笔 | 画笔工具，含颜色、粗细、笔迹效果 |
| 荧光笔 | 荧光笔工具，含颜色、透明度和粗细 |
| 橡皮擦 | 橡皮擦工具，点擦除/圈擦除/清空 |
| 选择 | 选择工具，框选/点选/套索 |
| 文本 | 文本输入和编辑 |
| 智能笔 | 智能形状识别 |
| **绘图工具** | 图形绘制（展开二级菜单） |
| **学科工具** | 学科教学工具（展开二级菜单） |
| **页面管理** | 多页操作（展开二级菜单） |
| **资源** | 媒体资源插入（展开二级菜单） |
| **更多** | 其他设置（展开二级菜单） |

## 5. 主画布区域

- 保持全屏显示，不受底部工具栏影响（工具栏悬浮或固定）
- 支持：
  - 手写笔迹 (InkCanvas)
  - 矢量图形 (ShapeCanvas)
  - 文本框
  - 媒体资源

## 7. 交互说明

1. **点击一级菜单**：展开对应二级菜单，其他菜单收起
2. **点击二级菜单项**：显示三级菜单（如果有）或执行操作
3. **点击画布**：收起所有菜单
4. **工具栏可以折叠/展开**：点击"收起"按钮最小化工具栏
