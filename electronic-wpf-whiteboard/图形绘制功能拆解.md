# 图形绘制功能拆解（详细步骤）

---

## 一、基础图形绘制

### 1.1 图形类别与实现步骤

#### 📏 直线
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 工具按钮点击 | 用户点击"直线"按钮 |
| 2 | 光标切换 | 鼠标光标变为十字形或画线图标 |
| 3 | 起点确定 | 按下鼠标左键，记录起点坐标 (x1, y1) |
| 4 | 实时预览 | 拖动时显示虚线预览 |
| 5 | 终点确定 | 释放鼠标，记录终点坐标 (x2, y2) |
| 6 | 创建图形 | 创建 Line 对象，添加到 ShapeCanvas |
| 7 | 更新状态 | 将图形添加到 CanvasState.Shapes |

#### 📐 射线 / 线段
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1-4 | 同直线 | 同上 |
| 5 | 终点确定后 | 射线：起点 + 方向；线段：固定两端点 |
| 6 | 创建图形 | 使用 Path 或 Line，标记类型为 Ray/Segment |

#### ⬜ 矩形
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 工具按钮点击 | 点击"矩形"按钮 |
| 2 | 展开子菜单 | 显示：矩形、正方形、圆角矩形 |
| 3 | 子类型选择 | 用户选择具体类型 |
| 4 | 起点确定 | 按下鼠标，记录左上角 (x, y) |
| 5 | 实时预览 | 拖动显示矩形轮廓 |
| 6 | 终点确定 | 释放鼠标，记录右下角 |
| 7 | 创建图形 | 创建 Rectangle，设置 CornerRadius(圆角) |
| 8 | 应用属性 | 设置 Fill/Stroke/StrokeThickness |

#### ⭕ 椭圆 / 圆
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 工具按钮点击 | 点击"椭圆"按钮 |
| 2 | 展开子菜单 | 显示：椭圆、圆 |
| 3 | 子类型选择 | 选择后锁定宽高比(圆)或自由(椭圆) |
| 4-7 | 同矩形 | 起点→预览→终点→创建 Ellipse |

#### 🔺 三角形
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 工具按钮点击 | 点击"三角形"按钮 |
| 2 | 展开子菜单 | 显示：一般三角形、等腰三角形、直角三角形 |
| 3 | 子类型选择 | 选择绘制模式 |
| 4 | 三点确定 | 按下记录第1点，拖动确定第2点，再次点击确定第3点 |
| 5 | 创建图形 | 使用 Path 构建三角形路径 |

#### ⬡ 多边形
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 工具按钮点击 | 点击"多边形"按钮 |
| 2 | 边数输入 | 弹出对话框输入边数(3-10)，或点击预设 |
| 3 | 顶点确定 | 依次点击各顶点，双击完成 |
| 4 | 闭合图形 | 自动连接最后一点与第一点 |
| 5 | 创建图形 | 使用 Path 构建多边形路径 |

#### ➤ 箭头
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1-5 | 同直线 | 绘制线条过程相同 |
| 6 | 添加箭头头部 | 在终点位置绘制箭头head |

#### --- 虚线
| 步骤 | 内容 | 说明 |
|------|------|------|
| 1-5 | 同直线 | 绘制过程相同 |
| 6 | 设置虚线样式 | StrokeDashArray = "4,2" |

### 1.2 图形属性设置

#### 颜色选择
```
步骤：
1. 点击图形选中
2. 显示属性面板（或右键菜单）
3. 点击"边框颜色" → 弹出颜色选择器
4. 选择预设颜色或自定义RGB
5. 应用到 ShapeModel.StrokeColor
6. 重新渲染图形
```

#### 填充设置
```
步骤：
1. 点击图形选中
2. 点击"填充颜色"
3. 选择：无填充 / 纯色 / 渐变
4. 应用到 ShapeModel.FillColor
5. 重新渲染
```

#### 边框粗细
```
步骤：
1. 选中图形
2. 拖动粗细滑块或输入数值
3. 应用到 ShapeModel.StrokeThickness
4. 重新渲染
```

#### 边框填充模式
```
选项：
- 仅边框 (Fill=None, Stroke=颜色)
- 仅填充 (Fill=颜色, Stroke=None)
- 边框+填充 (Fill=颜色, Stroke=颜色)

步骤：
1. 选中图形
2. 点击模式按钮
3. 应用对应属性组合
```

## 二、图形变换（选中后操作）

### 2.1 选中图形

#### 点选
```
步骤：
1. 点击"选择"工具
2. 点击画布上的图形
3. 显示选中高亮边框(Adorner)
4. 更新 CanvasState.SelectedShapeIds
```

#### 框选
```
步骤：
1. 选择工具下，鼠标左键拖动
2. 绘制虚线矩形框
3. 框内的图形全部选中
4. 可多选，按住Ctrl点击可追加/取消
```

### 2.2 移动图形

```
步骤：
1. 选中图形
2. 鼠标指针变为移动图标
3. 按下左键拖动
4. 实时更新图形位置 (ShapeModel.X/Y)
5. 拖动过程中显示辅助线(可选)
```

### 2.3 旋转图形

```
步骤：
1. 选中图形，显示旋转把手(Adorner)
2. 鼠标移动到旋转把手上，变为旋转图标
3. 按住左键拖动
4. 计算旋转角度：angleDelta = Vector.AngleBetween(startVector, currentVector)
5. 实时更新 ShapeModel.RotationAngle
6. 应用 RotateTransform 到图形
```

### 2.4 缩放图形

```
步骤：
1. 选中图形，显示8个缩放把手(上下左右+四角)
2. 鼠标移到把手上，变为双箭头
3. 按住左键拖动
4. 根据拖动方向更新 Width/Height
5. 保持宽高比(Shift键)或自由缩放
```

### 2.5 删除图形

```
步骤：
1. 选中图形(一个或多个)
2. 按 Delete 键 或 点击删除按钮
3. 从 CanvasState.Shapes 中移除
4. 从 Canvas 中移除图形元素
```

## 三、高级功能

### 3.1 图形吸附对齐 (Snap)

```
实现步骤：
1. 实现 ISnapService 接口
2. 对比当前图形与其他图形的：
   - CenterX / CenterY (中心对齐)
   - Left / Right / Top / Bottom (边缘对齐)
3. 距离阈值判断 (< 5px 时吸附)
4. 吸附时显示辅助虚线
5. 释放时应用对齐位置
```

### 3.2 图形布尔运算 (Clipper2)

```
步骤：
1. 选中两个图形
2. 选择运算类型：
   - 并集 (Union)：合并为一个图形
   - 交集 (Intersection)：保留重叠部分
   - 差集 (Difference)：减去重叠部分
   - 分割 (Clip)：按轮廓分割
3. 使用 Clipper2 库计算新路径
4. 创建新图形，删除原图形
```

### 3.3 图形连接点 (Connector)

```
实现步骤：
1. ShapeModel 定义 Connectors 列表
2. 每个 Connector 包含 RelativeX, RelativeY (0~1)
3. 渲染时计算绝对坐标并显示小圆点
4. 拖动另一图形靠近时，显示连接吸附提示
5. 连接后创建 ConnectionModel
6. 移动任一图形，另一图形跟随
```

## 四、Undo/Redo 支持

```
图形操作命令类：
- AddShapeCommand
- RemoveShapeCommand
- MoveShapeCommand
- RotateShapeCommand
- ScaleShapeCommand
- ChangePropertyCommand (颜色/填充等)

实现步骤：
1. 每一步操作创建对应 Command 对象
2. Command.Execute() 执行操作
3. Command.Undo() 撤销操作
4. 存入命令历史栈
5. Undo 时弹出栈顶，执行 Undo()
6. Redo 时重新执行 Execute()
```

## 五、数据模型

```csharp
public class ShapeModel
{
    public Guid Id { get; set; } = Guid.NewGuid();
    public ShapeType ShapeType { get; set; }  // Rectangle, Ellipse, Line, etc.
    
    // 位置和尺寸
    public double X { get; set; }
    public double Y { get; set; }
    public double Width { get; set; }
    public double Height { get; set; }
    
    // 旋转
    public double RotationAngle { get; set; }
    public double CenterX { get; set; }
    public double CenterY { get; set; }
    
    // 属性
    public string FillColor { get; set; } = "#FFFFFFFF";
    public string StrokeColor { get; set; } = "#FF000000";
    public double StrokeThickness { get; set; } = 2.0;
    
    // 连接点
    public List<ConnectorPoint> Connectors { get; set; } = new();
    
    // 分组
    public Guid? GroupId { get; set; }
}
```
