## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 |
|------|------|
| **简洁高效** | 界面简洁，操作高效，减少用户认知负担 |
| **直观易用** | 功能布局合理，交互符合直觉，学习成本低 |
| **专业精准** | 工具专业，支持精确操作，满足教学需求 |
| **灵活扩展** | 界面可定制，功能可扩展，适应不同场景 |
| **跨平台一致** | 不同平台体验一致，操作习惯统一 |

### 1.2 设计关键词

```
专业 · 教育 · 简洁 · 高效 · 直观 · 精准 · 友好
```

### 1.3 用户体验目标

| 目标 | 具体表现 |
|------|----------|
| 快速上手 | 新用户 5 分钟内掌握基本操作 |
| 高效操作 | 常用功能 2 次点击内可达 |
| 精确控制 | 支持精确数值输入和微调 |
| 无缝切换 | 不同模式/工具间切换流畅 |
| 零误操作 | 关键操作有确认，支持撤销 |

## 4. 组件规范

### 4.1 按钮组件

#### 4.1.1 按钮类型

| 类型 | 样式 | 用途 |
|------|------|------|
| **主要按钮** | 蓝色背景，白色文字 | 主要操作，提交确认 |
| **次要按钮** | 白色背景，蓝色边框 | 次要操作，取消返回 |
| **文字按钮** | 无背景边框 | 轻量操作，链接跳转 |
| **危险按钮** | 红色背景 | 删除、清空等危险操作 |
| **图标按钮** | 正方形，仅图标 | 工具栏按钮 |

#### 4.1.2 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| 小 | 28px | 8px 12px | 12px | 紧凑区域 |
| 中 | 32px | 12px 16px | 14px | 默认使用 |
| 大 | 40px | 16px 24px | 16px | 主要操作 |

#### 4.1.3 按钮状态

| 状态 | 样式变化 |
|------|----------|
| 默认 | 正常显示 |
| 悬停 | 背景色加深 10% |
| 按下 | 背景色加深 20% |
| 禁用 | 透明度 50%，鼠标禁用样式 |
| 加载 | 显示加载动画，禁用点击 |

/* 错误状态 */

| 操作 | 方式 | 说明 |
|------|------|------|

| 框选 | 拖拽空白区域 | 框选区域内对象 |
| 移动 | 拖拽 | 移动选中对象 |
| 复制 | Ctrl + 拖拽 | 复制并移动 |
| 旋转 | 拖拽旋转手柄 | 围绕中心旋转 |

## 8. 动效规范

### 8.1 过渡动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 面板展开/收起 | 高度变化 + 透明度 | 200ms |
| 弹窗显示/隐藏 | 缩放 + 透明度 | 150ms |
| 标签页切换 | 滑动 | 200ms |
| 下拉菜单展开 | 高度变化 | 150ms |

### 8.2 交互反馈动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 按钮点击 | 缩放 0.95 → 1 | 100ms |
| 图形选中 | 边框高亮 | 150ms |
| 吸靠提示 | 辅助线显示 | 100ms |
| 拖拽释放 | 轻微弹跳 | 200ms |

### 8.3 加载动效

| 场景 | 动效 |
|------|------|
| 页面加载 | 圆形旋转 Spinner |
| 按钮加载 | 圆形 Spinner + 按钮禁用 |
| 文件保存 | 进度条 |

### 8.4 动画曲线

```css
/* 标准曲线 */
transition-timing-function: ease;

/* 快出慢入（适合进入动画） */
transition-timing-function: ease-out;

/* 慢出快入（适合退出动画） */
transition-timing-function: ease-in;

/* 弹性曲线（适合强调动画） */
transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55);
```

## 9. 无障碍设计

### 9.1 键盘导航

- 所有交互元素可通过 Tab 键访问
- 提供可见的焦点指示器
- 支持 Enter/Space 激活按钮
- 支持 Escape 关闭弹窗

### 9.2 屏幕阅读器支持

- 所有图标提供 aria-label
- 图片提供 alt 文本
- 表单元素关联 label
- 动态内容使用 aria-live

### 9.3 颜色对比度

- 正文文字对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1
- 图标对比度 ≥ 3:1
- 提供高对比度模式（可选）

### 9.4 字体缩放

- 支持用户自定义字体大小
- 界面元素使用相对单位（rem/em）
- 测试 200% 放大效果
