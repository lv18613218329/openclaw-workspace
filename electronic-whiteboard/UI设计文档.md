## 1. 设计目标与原则

### 1.1 核心目标

打造一款**面向教师和学生的教学专用电子白板/黑板应用**，具备以下核心特点：

- 🎨 **直观易用**：工具触手可及，5分钟内可熟练操作
- 📚 **学科丰富**：支持数学、物理、化学等多学科教学工具
- 🎯 **高效教学**：减少技术操作时间，聚焦教学内容
- 🌈 **视觉舒适**：长时间使用不疲劳，保护视力

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **一致性** | 相同功能使用相同的交互方式，图标风格统一 |
| **层次分明** | 常用工具放显眼位置，次要功能收纳至二级菜单 |
| **即时反馈** | 所有操作都有视觉/触觉反馈 |
| **容错性** | 支持撤销/重做，误操作可恢复 |
| **无障碍** | 考虑大屏触控、手写笔等多种输入方式 |

## 2. 整体界面布局

### 2.1 布局概览

采用**四区布局**设计，参考希沃白板5的经典界面：

```
┌─────────────────────────────────────────────────────────────┐
│                      顶部工具栏 (48px)                       │
├──────┬────────────────────────────────────────────┬─────────┤
│      │                                            │         │
│ 侧   │                                            │  属性   │
│ 边   │              无限画布                      │  面板   │
│ 工   │           (核心绘图区域)                   │ (可选)  │
│ 具   │                                            │         │
│ 栏   │                                            │         │
│(72px)│                                            │ (280px) │
│      │                                            │         │
├──────┴────────────────────────────────────────────┴─────────┤
│                      底部状态栏 (32px)                       │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 布局尺寸规范

| 区域 | 宽度/高度 | 可调整性 |
|------|-----------|----------|
| 顶部工具栏 | 高度 48px | 固定 |
| 左侧工具栏 | 宽度 72px | 可折叠至 48px |
| 右侧属性面板 | 宽度 280px | 可隐藏/展开 |
| 底部状态栏 | 高度 32px | 固定 |
| 画布区域 | 自适应 | 填满剩余空间 |

### 2.3 响应式断点

- **标准模式**：≥ 1280px（完整四区布局）
- **紧凑模式**：1024px - 1279px（隐藏右侧面板，工具栏整合）
- **平板模式**：768px - 1023px（底部工具栏，简化布局）
- **手机模式**：< 768px（仅保留画布和浮动工具）

## 3. 顶部工具栏设计

### 3.1 工具栏区域划分

顶部工具栏从左到右分为以下功能区：

```
┌──────────────────────────────────────────────────────────────────┐
│ [Logo] [新建][打开][保存] │ [撤销][重做] │ [选择][缩放][平移] │ [更多]│
└──────────────────────────────────────────────────────────────────┘
```

| 区域 | 功能组 | 包含内容 |
|------|--------|----------|
| 左侧 | 文件操作 | Logo、新建、打开、保存、导出 |
| 中左 | 编辑操作 | 撤销、重做 |
| 中部 | 视图操作 | 选择、缩放、平移、画布模式切换 |
| 右侧 | 系统设置 | 更多设置（主题、语言、帮助） |

### 3.2 工具栏按钮规范

- **按钮尺寸**：36px × 36px
- **图标尺寸**：20px × 20px
- **间距**：8px
- **悬停状态**：背景色加深 10%，显示 tooltip
- **按下状态**：背景色加深 20%
- **禁用状态**：opacity: 0.5，cursor: not-allowed

### 3.3 顶部工具栏功能详情

| 图标 | 名称 | 快捷键 | 功能说明 |
|------|------|--------|----------|
| 🆕 | 新建 | Ctrl+N | 创建新白板，提示是否保存当前 |
| 📂 | 打开 | Ctrl+O | 打开已有白板文件 (.ewb) |
| 💾 | 保存 | Ctrl+S | 保存当前白板 |
| 📤 | 导出 | Ctrl+E | 导出为 PNG/PDF |
| ↩️ | 撤销 | Ctrl+Z | 撤销上一步操作 |
| ↪️ | 重做 | Ctrl+Y | 重做已撤销的操作 |
| 👆 | 选择 | V | 选择、移动、调整图形 |
| 🔍 | 缩放 | Z | 放大/缩小画布 |
| ✋ | 平移 | H | 拖动画布平移 |
| 🎭 | 模式 | - | 切换白板/黑板/课件模式 |

## 7. 工具分类与图标设计

### 7.1 图标设计原则

- **风格**：扁平化、简约、辨识度高
- **线条粗细**：2px 统一线条
- **圆角**：4px 统一圆角
- **颜色**：主色 #333333，辅色 #666666
- **尺寸**：20×20px（工具栏）、24×24px（工具面板）

### 7.2 图标色彩编码

| 分类 | 主色调 | 用途 |
|------|--------|------|
| 基础图形 | 蓝色 #2196F3 | 默认状态 |
| 尺规工具 | 绿色 #4CAF50 | 测量类工具 |
| 数学工具 | 紫色 #9C27B0 | 数学相关 |
| 物理工具 | 橙色 #FF9800 | 物理相关 |
| 化学工具 | 青色 #00BCD4 | 化学相关 |
| 公式工具 | 红色 #F44336 | 公式/文本 |
| 手写工具 | 棕色 #795548 | 手写/绘画 |

### 7.3 工具选中状态

- 选中工具：背景色 #E3F2FD，图标颜色加深
- 工具组展开：左边框显示 3px 分类主题色
- 常用工具标记：右上角小星星 ⭐
