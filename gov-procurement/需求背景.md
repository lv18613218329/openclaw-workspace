# 需求背景

## 业务场景

公司业务覆盖教育行业，需要及时获取全国各地学校相关的政府采购招标信息，用于：

1. **商机发现**：及时发现潜在客户的采购需求
2. **市场分析**：了解行业动态和竞争情况
3. **客户拓展**：获取学校联系方式和采购偏好

## 数据来源

- **网站**：中国政府采购网（https://search.ccgp.gov.cn/bxsearch）
- **权威性**：财政部唯一指定政府采购信息网络发布媒体
- **数据量**：全国范围内的政府采购公告

## 采集目标

### 关键词配置（5大类23个）

| 类别 | 关键词 |
|------|--------|
| 政策项目类 | 双优校建设、双高建设、教师档案袋 |
| 智慧校园类 | 职业教育SaaS、软件平台、云服务、智慧校园、教务系统、信息化标杆校、学工系统、数据中台 |
| 领讲产品类 | 智慧教室、智能讲台、教学软件、教学平台、多媒体教室、教室改造 |
| 课程资源类 | 资源、精品课程、开放课程、微课资源、数字人课程、数字人视频、教学视频 |
| 师资培训类 | AI、数智化素养、数字素养、师资培训、人工智能培训、赋能培训 |

### 学校关键词过滤规则

采购人名称必须包含以下关键词之一才入库：

```
学校、中心、学院、中学、高中、集团、大学、小学
```

**示例**：
- ✅ 赤峰建筑工程**学校** → 有效
- ✅ 当阳市职业技术教育**中心** → 有效
- ✅ 哈尔滨**大学** → 有效
- ❌ 某某科技公司 → 无效（不包含学校关键词）

## 数据质量要求

| 字段 | 要求 | 校验规则 |
|------|------|---------|
| 标题 | 完整标题，不截取 | 长度 ≥ 10字符 |
| URL | 真实可访问地址 | 以 `http://www.ccgp.gov.cn/` 开头，包含日期路径 |
| 采购人 | 完整名称 | 包含学校关键词 |
| 地区 | 省市名称 | 有效的行政区划名称 |
| 发布日期 | YYYY-MM-DD | 有效日期格式 |

## 时间范围

- **默认**：最近 30 天
- **支持**：自定义时间范围

## 数据用途

1. 导入 CRM 系统进行客户跟进
2. 生成市场分析报告
3. 同步到飞书多维表格供团队协作
