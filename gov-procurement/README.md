# 政府采购信息采集工具

从中国政府采购网（https://search.ccgp.gov.cn/）自动采集招标公告信息，支持关键词过滤、学校关键词筛选、增量同步。

## 快速开始

```bash
# 1. 进入项目目录
cd ~/.openclaw/workspace/gov-procurement

# 2. 安装依赖（首次运行）
npm install

# 3. 执行采集
node crawl-v3.js

# 4. 查看数据
node show-data.js
```

## 一分钟了解

**采集什么**：政府采购网的招标公告信息

**采集条件**：
- 关键词：智慧校园、教学平台、双优校建设等 23 个
- 时间范围：最近 30 天
- 过滤规则：采购人必须包含学校关键词（学校、中心、学院、中学、高中、集团、大学、小学）

**采集字段**：
- 标题（完整）
- URL（真实可访问）
- 采购人
- 地区
- 发布日期
- 公告类型
