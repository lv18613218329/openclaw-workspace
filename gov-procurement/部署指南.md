# 政府采购信息采集工具 - 部署指南

## 第一步：解压文件

将 `gov-procurement.zip` 解压到以下位置：

**Windows:**
```
C:\Users\{你的用户名}\.openclaw\workspace\gov-procurement
```

**Mac/Linux:**
```
~/.openclaw/workspace/gov-procurement
```

---

## 第二步：安装 Skill（让 OpenClaw 识别此工具）

将 `skill` 文件夹复制到 OpenClaw 的 skills 目录：

**Windows:**
```
复制 skill 文件夹到：
C:\Users\{你的用户名}\AppData\Roaming\npm\node_modules\openclaw\skills\gov-procurement
```

**Mac/Linux:**
```
复制 skill 文件夹到：
~/.npm/lib/node_modules/openclaw/skills/gov-procurement
```

---

## 第三步：安装依赖

```bash
cd ~/.openclaw/workspace/gov-procurement
npm install
```

---

## 第四步：运行采集

确保 OpenClaw 已启动，浏览器已打开并激活 Browser Relay 扩展，然后：

```bash
node crawl-v3.js
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `node crawl-v3.js` | 执行采集 |
| `node show-data.js` | 查看数据 |
| `node check-quality.js` | 检查数据质量 |

---

## 功能说明

- 自动采集政府采购网招标公告
- 按关键词搜索（智慧校园、教学平台等23个）
- 自动过滤学校客户数据
- 支持增量同步，自动去重
