# 政府采购信息采集 - 改进方案 v2

## 一、问题分析

### 1.1 当前存在的问题

| 问题 | 描述 | 影响 |
|------|------|------|
| **标题不完整** | AI解析时截取了部分标题，非完整标题 | 无法准确匹配业务需求 |
| **URL地址错误** | AI编造了部分URL，无法正常访问 | 无法查看详情页 |
| **地区未提取** | 部分记录的地区字段为空 | 无法按地区筛选 |

### 1.2 问题根因

1. **解析方式问题**：使用正则表达式解析HTML，对页面结构依赖强，容易出错
2. **数据校验缺失**：未对解析结果进行有效性校验
3. **过滤策略问题**：先采集后过滤，导致无效数据入库

## 二、改进方案

### 2.1 优化查询策略

**原策略**：按关键词查询 → 解析所有结果 → 入库

**新策略**：按关键词查询 → 过滤无效数据 → 解析有效数据 → 校验后入库

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 关键词+时间  │ → │ 学校关键词   │ → │ AI解析字段  │ → │ 校验后入库  │
│ 查询数据     │    │ 过滤        │    │             │    │             │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### 2.2 学校关键词过滤规则

**有效数据判断条件**（满足任一即可）：

采购人名称包含以下关键词：
- 学校
- 中心
- 学院
- 中学
- 高中
- 集团
- 大学
- 小学

**示例**：
- ✅ `赤峰建筑工程学校` → 有效
- ✅ `当阳市职业技术教育中心` → 有效
- ✅ `某某大学` → 有效
- ❌ `某某公司` → 无效（不包含学校关键词）
- ❌ `某某研究院` → 无效（不包含学校关键词）

### 2.3 数据解析要求

| 字段 | 要求 | 校验规则 |
|------|------|---------|
| **标题** | 完整标题，不得截取 | 长度 ≥ 10字符，不得以`...`结尾 |
| **URL** | 完整可访问地址 | 必须以 `http://www.ccgp.gov.cn/` 开头 |
| **地区** | 完整省市名称 | 必须为有效行政区划名称 |
| **采购人** | 完整名称 | 长度 ≥ 4字符 |
| **发布日期** | YYYY-MM-DD格式 | 有效日期 |

## 三、技术实现方案

### 3.1 采集流程

```javascript
// 伪代码
async function crawl(keyword, startDate, endDate) {
  // 1. 搜索页面获取原始列表
  const searchResults = await search(keyword, startDate, endDate);
  
  // 2. 过滤：只保留采购人包含学校关键词的记录
  const validResults = searchResults.filter(item => {
    const buyer = item.buyer || '';
    return schoolKeywords.some(kw => buyer.includes(kw));
  });
  
  // 3. 解析每条有效记录的详细信息
  for (const item of validResults) {
    // 3.1 访问详情页获取完整信息
    const detail = await fetchDetailPage(item.url);
    
    // 3.2 AI解析详情页内容
    const parsed = await aiParse(detail, {
      requirements: {
        title: '完整标题，不得截取',
        url: '原始URL，必须能访问',
        region: '完整地区名称'
      }
    });
    
    // 4. 校验解析结果
    if (validate(parsed)) {
      save(parsed);
    }
  }
}
```

### 3.3 数据校验函数

```javascript
function validateRecord(record) {
  const errors = [];
  
  // 标题校验
  if (!record.title || record.title.length < 10) {
    errors.push('标题过短或为空');
  }
  if (record.title.endsWith('...')) {
    errors.push('标题被截取');
  }
  
  // URL校验
  if (!record.url || !record.url.startsWith('http://www.ccgp.gov.cn/')) {
    errors.push('URL格式错误');
  }
  
  // 地区校验
  if (!record.region || record.region.length < 2) {
    errors.push('地区为空');
  }
  
  return {
    valid: errors.length === 0,
    errors
  };
}
```
