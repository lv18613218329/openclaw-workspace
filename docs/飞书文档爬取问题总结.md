# 飞书文档爬取问题总结

## 任务目标
从飞书 Wiki 页面 (`https://o08w2nllqni.feishu.cn/wiki/WamWwuKhBiClHJknRkecRdQnnTc`) 抓取完整内容并导出为 Word/PDF 文档，同时保留排版和图片。

---

## 遇到的问题

### 1. 虚拟滚动 (Virtual Scrolling)
- **问题**: 飞书文档采用虚拟渲染，内容不会一次性加载到 DOM 中
- **现象**: 页面高度显示 900px，但实际内容需要滚动到 50000+px 才能全部加载
- **影响**: 爬取的 HTML 是不完整的，只包含可视区域的内容
- **解决方案**: 
  - 手动滚动到页面底部触发加载
  - 或使用脚本模拟滚动

### 2. 打印保护机制
飞书文档有多重打印保护：

#### 2.1 CSS 媒体查询拦截
```css
@media print {
  body { display: none !important; }
}
```
- **影响**: 直接调用 `window.print()` 或浏览器打印功能时，页面内容为空
- **尝试的绕过方式**:
  - 注入 CSS 覆盖 `body { display: block !important; }` - 失败
  - 通过 CDP (Chrome DevTools Protocol) 直接调用 `Page.printToPDF` - 失败，输出仍为空

#### 2.2 打印禁止占位符
```html
<div class="gpf-biz-action-manager-forbidden-placeholder" data-type="print-forbidden-placeholder">
  附件不支持打印
</div>
```
- **现象**: 嵌入的 Excel 文件、视频等附件显示"不支持打印"

#### 2.3 权限拦截
- **现象**: 尝试打印时弹出 "没有权限" 的提示
- **原因**: 飞书在打印事件上监听了权限检查

### 3. 认证拦截
- **问题**: `web_fetch` 工具无法直接获取页面内容
- **原因**: 页面有登录验证，302 重定向到登录页
- **解决方案**: 使用 OpenClaw 的浏览器控制 (CDP)

### 4. 导出功能隐藏
- **位置**: 飞书 Wiki 页面的导出功能入口较深
- **尝试**: 
  - 点击右上角「更多」按钮 → 菜单中没有导出选项
  - 尝试右键文件名 → 未找到有效入口

---

## 尝试过的方案

| 方案 | 结果 | 说明 |
|------|------|------|
| `web_fetch` 直接抓取 | ❌ 失败 | 登录拦截，302 重定向 |
| 浏览器模拟滚动 + 获取 HTML | ⚠️ 部分成功 | 只能获取已渲染内容，虚拟加载部分丢失 |
| 注入 CSS 移除打印保护 | ❌ 失败 | 飞书有更深层的权限检查 |
| `openclaw browser pdf` | ❌ 失败 | CDP 打印仍被拦截，输出为空 |
| `chrome-remote-interface` CDP | ❌ 失败 | 同样的拦截问题 |
| 手动浏览器打印 | ❌ 失败 | 弹窗提示"没有权限" |
| 完整滚动后截图 | ✅ 部分成功 | 只能截可见区域，非完整文档 |
| Wiki 转 Docx 链接 | ❌ 失败 | 链接无效，页面不存在 |

1. **虚拟滚动**: 需要找到飞书加载数据的 API，直接调用获取完整 JSON 内容
2. **打印保护**: 需要绕过飞书的打印权限检查
3. **官方导出**: 需要找到正确的导出入口或 API

## 可行的替代方案

### 方案 1: 手动导出 (推荐)
1. 进入知识库页面
2. 在文档列表中右键该文档
3. 选择「导出」→ 「导出为 Word」
4. Word 打开后另存为 PDF

### 方案 2: 复制粘贴
1. 在浏览器中完整加载文档（滚动到底部）
2. `Ctrl + A` 全选
3. `Ctrl + C` 复制
4. 粘贴到 Word/编辑器

### 方案 3: 飞书 API (需配置)
通过飞书开放平台 API 获取文档内容：
- 需要配置 App ID 和 App Secret
- 需要文档的读取权限
- 可获取 JSON 格式的文档结构

## 技术要点

### OpenClaw 浏览器控制命令
```bash
# 打开页面
openclaw browser open <url>

# 截图
openclaw browser screenshot
openclaw browser screenshot --full-page

# 打印 PDF (当前被拦截)
openclaw browser pdf

# 注入 JS
openclaw browser evaluate --fn "<js code>"

# 查看元素
openclaw browser snapshot
```

### CDP 连接 (用于高级自动化)
- 端口: `18800`
- 可用库: `chrome-remote-interface`, `puppeteer-core`
